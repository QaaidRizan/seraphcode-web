## Alternative: Use Unsplash Direct Embed URLs (No Download Needed)

If you'd prefer to reference images directly without downloading (useful for prototyping), you can use the Unsplash CDN format:

```
https://images.unsplash.com/photo-{PHOTO_ID}?w=1200&q=80&auto=format&fit=crop
```

Example in Next.js:
```jsx
<Image
  src="https://images.unsplash.com/photo-1461749280684-dccba630e2f6?w=1200&q=80&auto=format&fit=crop"
  alt="SaaS development dashboard with code"
  width={1200}
  height={600}
/>
```

> ⚠️ **Note:** For production, downloading and hosting in your own `/public/assets/` is recommended for performance, reliability, and to avoid external dependency.

## Recommended Approach for Your Next.js Site

Since SeraphCode uses Next.js (`next/image`), here's the best approach:

1. **Download** all images from the links above
2. **Place** them in `/public/assets/images/`
3. **Reference** them as `/assets/images/saas-development.jpg` (no `/public` prefix needed in Next.js)
4. Use `next/image` for automatic optimisation:

```jsx
import Image from 'next/image'

// Replace this:
<img src="/assets/1213.jpg" />

// With this:
<Image
  src="/assets/images/saas-development.jpg"
  alt="SaaS development architecture diagram"
  width={1200}
  height={600}
  className="your-existing-class"
/>
```
